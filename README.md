# BÁO CÁO ASSIGNMENT 7: TRIỂN KHAI SCRIPT AUTOMATION CHO CHỨC NĂNG CÁ NHÂN

Báo cáo này tổng hợp kết quả triển khai các kịch bản kiểm thử tự động (Automation Scripts) cho website mục tiêu **Rabity.vn**

---

## I. MỤC TIÊU & YÊU CẦU DỰ ÁN

* **Website kiểm thử (SUT):** `https://rabity.vn/`
* **Công cụ Automation:** [TÊN CÔNG CỤ ĐÃ CHỌN: Ví dụ: **Playwright**]
* **Mục tiêu:** Mỗi sinh viên triển khai tự động hóa cho **2 chức năng** đã được phân công.
* **Quy tắc đặt tên:** Mỗi thành viên đóng góp 1 file script riêng theo định dạng: `MaSV_HoTen_Lab7.[ext]`
